    class JSONSchema::CompositeValidator

    • JSONSchema::CompositeValidator
    • Reference
    • Object

    Overview

    Validates schema that has any of the keywords for composite/compound schema set as a property. The possible keywords are allOf, anyOf, oneOf, and not.

    This is a raw Validator class that you most likely do not need to use directly. See the JSONSchema#create_validator macro for common usage of this shard.
